# 房屋租赁管理系统 (House Rental Management System)

基于 .NET 8 的 Windows Forms 桌面应用程序，提供完整的房屋租赁管理功能。

## 项目简介

本系统是一个 C/S 架构的房屋租赁管理平台，包含房源管理、租户管理、用户管理、订单管理等功能模块，采用三层架构（表示层 - 业务逻辑层 - 数据访问层）设计。

## 技术栈

- **框架**: .NET 8 (Windows Forms)
- **语言**: C#
- **数据库**: SQL Server
- **第三方库**: NPOI 2.7.4 (Excel 导入导出), IrisSkin4 (界面美化)

## 项目结构

```
房屋租赁系统/
├── 房屋租赁系统/          # 表示层 (WinForms UI)
│   ├── 房屋管理/          # 房源增删改查
│   ├── 房屋类型管理/       # 房源分类管理
│   ├── 用户管理/          # 用户增删改查、注册
│   ├── 租户管理/          # 租户信息管理
│   ├── 订单管理/          # 租赁订单管理
│   └── 反馈信息/          # 用户反馈
├── 房屋租赁系统.BLL/      # 业务逻辑层
├── 房屋租赁系统.DAL/      # 数据访问层
└── 房屋租赁系统.Models/   # 数据模型层
```

## 功能模块

- **房屋管理**: 房源添加、编辑、删除、条件查询
- **房屋类型管理**: 房源分类维护
- **用户管理**: 系统用户 CRUD 及权限控制
- **租户管理**: 租户信息登记与维护
- **订单管理**: 租赁订单创建与管理
- **用户注册**: 新用户自助注册
- **反馈信息**: 用户意见反馈收集
- **Excel 导入导出**: 基于 NPOI 的数据批量处理

## 注意事项

- 本项目使用 SQL Server 数据库，连接字符串配置在 `房屋租赁系统.DAL/sqlHelper.cs` 中
- 数据库名称为 `myHouses`
- 运行前需确保 SQL Server 服务可用，并根据实际环境修改连接字符串

## 开发环境

- Visual Studio 2022 (v17.13+)
- .NET 8.0 SDK
- SQL Server (LocalDB 或完整版)

## 架构说明

采用经典三层架构：

- **表示层 (UI)**: WinForms 窗体，负责用户交互
- **业务逻辑层 (BLL)**: 处理业务规则和逻辑
- **数据访问层 (DAL)**: 封装数据库操作，使用 ADO.NET + SqlHelper
- **模型层 (Models)**: 数据实体定义
